Welcome to the Ecology and Conservation of Migratory Birds

This website will be our home base for the R portion of the course. It will contain all of the R exercises and includes links to help files and data that we will use throughout the course.

Getting Started

R has become the universal quantitative interface for ecologists – it’s free, powerful, flexible enough to do everything but write manuscripts for you, and is relatively user-friendly. To get the most from this course, we strongly suggest completing a select set of self-guided learning exercises for R in the SWIRL R-package. Instructions for using SWIRL are located here and include the list of courses we would like you to complete prior to arrival. If you haven't already, be sure to:

  • Download and install the current version of R (available here)
  • Download and install R Studio (available here)
